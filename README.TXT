    operations such as searching for cars, adding/deleting cars, 
    displaying the details of cars, etc.

HOW TO START THIS PROJECT: create an object of ControlCenter class
AUTHORS: Miao Wang
USER INSTRUCTIONS:After user exit the system,
    just delete the ControlCenter object and create a new one
    to restart.
